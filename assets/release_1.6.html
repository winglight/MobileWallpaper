Daily Mobile Wallpaper | 手机壁纸大全

Version 1.6 | Release notes

Attention please: for our wallpaper is Live Wallpaper, you need to set Live Wallpaper manually that's always available once you did it. Here's how-to-set-livewallpaper-on-android.

What's new:

- Fix some bugs

v1.5 features:

- Set the default wallpaper of tips before picking up any image

v1.4 features:

- Add page down/up function by buttons of volume

v1.3 features:

- Share your favorite wallpapers with your friends by MMS, Email, Bluetooth, etc.

v1.2 features:

- Fix a bug.

v1.1 features:

- Long press on any image and then select context menu to set the image as live wallpaper.

v1.0 features:

- New feature to store "My Favorite" on the top link of every page，and long press on any hyper-link and then select context menu to "Add this link to my favorite"

- Favorite web page was split to three pages for different type of download; All - all of download contents; Html - saved links; Picture - download pictures

- Long press on E-mail address to open E-mail client

- After downloading an APK file on CheckUpdate page, package installer will be started and install the apk

- New menu item to browse downloaded content

- Auto translate pages into english while local language is english

- Long press on any hyper-link and then select context menu to save this link(including web pages, pictures, APK, etc), or to save the link including next level link(including web pages, pictures)

- Long press on any image and then select context menu to download this single image, or to download all images on this web page

- Downloaded content prior to be stored in sdcard, of which top folder is /sdcard/download, and content saved from link is put in /sdcard/download/html, and image downloaded is put in /sdcard/download/pic

请注意:本软件壁纸为Live wallpaper类型,需要手工设置系统的Live wallpaper为"手机壁纸大全"才可生效,这个设置只需要进行一次. 如何设置Live Wallpaper,可以参考这里.

最新特性：

- 修改bug

v1.5 特性

- 在没有设置任何图片作为壁纸前，使用默认提示作为桌面

v1.4 特性

- 增加使用音量键翻页的功能

v1.3 特性

- 通过彩信,蓝牙,Email等与朋友分享壁纸

v1.2 特性

- 修改bug

v1.1 特性

- 长按任意图片, 在弹出菜单中可以选择设置当前图片为live wallpaper

v1.0 特性

- 添加网络收藏功能，在页面上方的链接中，选择"我的收藏"即可查看，在长按任意链接弹出的菜单中选择"收藏这个链接"即可把链接添加到"我的收藏"中

- 收藏页面分成了三个不同类型的页面:All - 全部; Html - 下载链接; Picture - 下载图片

- 长按E-mail地址，可以直接打开E-mail客户端发送邮件

- 新增在更新页面下载APK后自动启动安装器安装软件

- 菜单中增加收藏功能,可以查看以往下载的内容

- 增加当本地语言为英语时自动翻译网页内容为英文的功能

- 长按任意链接, 在弹出菜单中可以选择下载链接内容(可以是页面、图片、APK等等格式), 或者下载链接页面及其下一级链接(包括页面、图片内容)

- 长按任意图片, 在弹出菜单中可以选择下载当前图片, 或者下载当前页面所有图片

- 下载内容优先保存在SDCARD上, 总目录位置在:/sdcard/download, 其中链接下载内容在/sdcard/download/html, 图片下载内容在/sdcard/download/pic
